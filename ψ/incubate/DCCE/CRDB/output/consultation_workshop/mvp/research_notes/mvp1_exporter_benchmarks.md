# MVP-1 (Exporter) Benchmarking: Real-World Relatives

## 1. Identified Relatives
*   **UKCP (UK Met Office)**: Bridges national scenarios to 12km local grids using "Summary Briefings" (Level 1) and "Online Viewers" (Level 2).
*   **Climate-ADAPT (EEA)**: Uses "Overview Reports" for regional narratives and "Interactive Maps" for sector-specific local risks.
*   **Cal-Adapt (California)**: Structures from statewide "Dashboards" to county-level "Interactive Tools."
*   **World Bank CCKP**: Provides "Country Narratives" (Level 1) vs "Subnational Data Downloaders" (Level 2).

## 2. Design Patterns (Level 1 vs Level 2)
| Feature | Level 1: Tactical Scoping | Level 2: Technical Design |
|---------|---------------------------|---------------------------|
| **Format** | Narrative Briefings, Static Summaries | Interactive Maps, Gridded Data API |
| **Resolution** | National/Regional (Provincial) | Local Grid (12km) / Site-specific |
| **Goal** | Policy Framing, Prioritization | Engineering Spec, Site Design |
| **User** | Senior Management, Policy Makers | Technical Officers, Engineers |

## 3. Confidence Score: 5/5
**Rationale**: Direct "Real-World Relatives" found that explicitly bridge narratives and data. The "Level 1 vs Level 2" framing is strongly mirrored in UKCP and Climate-ADAPT.
