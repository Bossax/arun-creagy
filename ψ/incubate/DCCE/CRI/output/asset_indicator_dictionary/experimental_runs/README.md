# Experimental Runs Folder

This folder stores outputs for the CRI Phase 2 v4 activation-linkage experiments.

## Initialized scaffold

Run-independent benchmark artifacts:
- `00_benchmark_11-concept-working-matrix.md`
- `00_scoring-rubric.md`

Per-experiment folders:
- `experiment_A/`
- `experiment_B/`
- `experiment_C/`

Each experiment folder is initialized with placeholders for:
- round 1 reuse-first log
- round 2 controlled-refinement log
- round 3 prune-and-stabilize log
- asset-chain worksheet
- concept-addition register
- evidence-decision log
- pruning-decision log
- final experiment summary

Comparison folder:
- `comparison/final-comparison-memo.md`

## Status boundary

This scaffold is an initialization surface only.
No Experiment A, B, or C execution results are stored here yet.

## Structural rule

The benchmark base for all experiments is fixed to:
- the six-pillar governance kernel from `CRI_Capacity_Tagging_Dictionary_v3.md`
- the 11 hardened asset concepts from `CRI_Asset_Concept_Summary_v1.md`
- the shared scoring rubric initialized in this folder
