## Lessons Learned
- **Dasymetric Bias**: Population density is a proxy for *exposure*, but history is the proxy for *probability*. Multiplying them is the only way to map true *risk*.
- **Shapefile Field Limits**: Always use explicit 10-character names for SHP columns to prevent the GDAL "laundered field name" errors.
