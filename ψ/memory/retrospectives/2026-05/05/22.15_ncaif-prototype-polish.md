## Lessons Learned
- **The "Guardrail" UI Pattern**: In climate services, showing "Not Recommended" (Red) is as important as showing "Ready" (Green). The 4-tier matrix approach is more effective than a single-selection filter because it forces the user to see the limitations of the data product across the entire administrative lifecycle.
- **Ergonomic Data Density**: 70% width is the sweet spot for data-heavy dashboards on 1080p+ screens.
