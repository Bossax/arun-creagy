# When a stakeholder workshop includes mixed technical literacy, do not frame the 

When a stakeholder workshop includes mixed technical literacy, do not frame the main generative session as open-ended co-creation from a blank page. A better pattern is to present a small set of prepared service or MVP concepts, ask participants to react to them, allow them to add missing services or product ideas, and then force cross-group learning through a structured peer-review round. This preserves realism, creativity, comparability, and peer learning. The strongest minimum concept frame for each proposed service/product is: intended users, problem or task supported, product form, required data or systems, and main implementation barrier.
