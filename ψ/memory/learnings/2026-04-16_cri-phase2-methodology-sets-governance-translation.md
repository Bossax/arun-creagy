# Learning: SETS-to-Governance Translation in CRI Phase 2 Methodology

In CRI Phase 2, the strongest framing is a deliberate dual-layer design:

1. Use SETS as the scientific umbrella to ensure coverage across social, ecological, technological, and institutional domains.
2. Use Institutional Readiness governance functions as the operational representation because these are the levers decision-makers can budget, assign, and improve.

This prevents two failure modes:

- Conceptual under-coverage (if governance is used without systems-level completeness checks).
- Operational paralysis (if full systems framing is used without administratively legible implementation levers).

Method document implication: when narrowing from broad frameworks to an implementation lens, the lineage must be stated explicitly (scan -> umbrella -> zoom-in), or the final framing can look arbitrary.
