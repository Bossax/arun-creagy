# When a governance-linkage taxonomy is under question, bounded comparative experi

When a governance-linkage taxonomy is under question, bounded comparative experimentation can outperform both rigid reuse and loose exploratory growth. In the CRI Phase 2 v4 session, a strict no-growth run preserved kernel purity but weakened bottleneck clarity for soft community-mediated asset cases, while an exploratory run temporarily widened the search space but pruned back to the same narrow result. The balanced condition proved strongest: preserve the main kernel, allow tightly bounded supplementation, then enforce aggressive pruning. If both conservative and exploratory runs converge on the same narrow supplement, that supplement is more trustworthy than either a rigid no-growth stance or an open-ended expansion stance.
