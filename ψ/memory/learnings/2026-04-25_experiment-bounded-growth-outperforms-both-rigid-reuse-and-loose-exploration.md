# Experiment-bounded growth outperforms both rigid reuse and loose exploration in linkage-taxonomy decisions

When testing whether a governance taxonomy needs to grow, the strongest method is a bounded comparative program rather than a single-pass rewrite. In this CRI Phase 2 v4 session, strict reuse-only logic preserved purity but weakened bottleneck clarity for the two softest asset families, while exploratory growth temporarily widened the search space but pruned back to the same narrow outcome. The most reliable result came from the balanced condition: preserve the main kernel, permit tightly bounded experimentation, and require aggressive pruning before adoption. This produced a stable outcome in which only one thin overlay concept survived and only for the two asset families that genuinely needed it. The pattern suggests a practical design rule: if both conservative and exploratory experiments converge on the same narrow supplement, the supplement is more trustworthy than either a rigid no-growth stance or a loose open-ended expansion stance.
