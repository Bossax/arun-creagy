## 3.3 แกนฉากทัศน์และชุดตัวบ่งชี้

การออกแบบฉากทัศน์ในรายงานฉบับนี้ตั้งอยู่บนความไม่แน่นอนสำคัญสองแกนที่มีผลโดยตรงต่อชีวิตของเด็กและเยาวชนในพื้นที่สามจังหวัดชายแดนภาคใต้ แกนแรกคือรูปแบบและการเชื่อมต่อของเส้นทางการเรียนรู้ แกนที่สองคือโครงสร้างเศรษฐกิจและเส้นทางรายได้ของครัวเรือน ทั้งสองแกนเชื่อมโยงกับหลักฐานจากสถานการณ์ปัจจุบันและแนวโน้มที่สังเคราะห์จากสัญญาณในบทที่ 2 และสะท้อนความตึงเครียดระหว่าง “ระบบที่ทำให้การอยู่ในเส้นทางการเรียนรู้เป็นไปได้จริงสำหรับเด็กกลุ่มเป้าหมาย” กับ “ระบบที่ยังผลักต้นทุนให้ครัวเรือนรับเอง”

บนแกนการเรียนรู้ รายงานเปรียบเทียบระหว่างโลกที่ระบบการศึกษาและการรับรองสมรรถนะยังคงยึดเส้นทางแบบเดียวและมาตรฐานส่วนกลางเป็นหลัก กับโลกที่ระบบยอมรับความหลากหลายของเส้นทางการเรียนรู้ และทำให้เส้นทางเหล่านั้นเชื่อมต่อกันได้จริง เด็กสามารถย้ายจากการเรียนในโรงเรียนสู่การเรียนรู้ที่เชื่อมกับศาสนา ชุมชน หรือการฝึกอาชีพ แล้วกลับเข้าสู่การรับรองสมรรถนะได้ โดยไม่ถูกตัดสิทธิ์จากโอกาสในอนาคตเพียงเพราะเลือกเส้นทางที่ต่างออกไป

บนแกนเศรษฐกิจ รายงานเปรียบเทียบระหว่างโลกที่เศรษฐกิจของพื้นที่ยังพึ่งพางานมูลค่าเพิ่มต่ำ รายได้ผันผวน และครัวเรือนต้องรับความเสี่ยงไว้เอง กับโลกที่เศรษฐกิจใหม่ที่เชื่อมโยงกับฮาลาล เศรษฐกิจสร้างสรรค์ และดิจิทัลเติบโตบนฐานทุนวัฒนธรรมและความเชื่อมโยงภูมิภาค ทำให้เกิดงานที่มีศักดิ์ศรีและรายได้ที่มั่นคงขึ้น และทำให้ครัวเรือนมีพื้นที่ในการลงทุนด้านการเรียนรู้ของเด็กมากขึ้น ความเปลี่ยนแปลงของแกนนี้จึงไม่ได้หมายถึงการเติบโตของตัวเลขเศรษฐกิจเพียงอย่างเดียว แต่หมายถึงรูปแบบการกระจายโอกาสว่าครัวเรือนที่มีความเปราะบางเข้าถึงได้จริงหรือไม่

เมื่อวางสองแกนนี้ตัดกัน จะได้ฉากทัศน์สี่แบบที่ใช้เป็นกรอบหลักของบทที่ 3

|  | เศรษฐกิจยังเปราะบางและพึ่งพางานมูลค่าเพิ่มต่ำ | เศรษฐกิจยกระดับบนฐานทุนวัฒนธรรมและดิจิทัล (กระจายโอกาสมากขึ้น) |
|---|---|---|
| ระบบการเรียนรู้แบบทางเดียวและการรับรองที่ยึดมาตรฐานส่วนกลางเป็นหลัก | ฉากทัศน์ 1 กรอบเดียว โอกาสติดหล่ม | ฉากทัศน์ 2 โตแต่ไหลออก |
| ระบบการเรียนรู้แบบหลากหลายที่เชื่อมต่อได้และการรับรองสมรรถนะหลายเส้นทาง | ฉากทัศน์ 3 ยืนบนราก รายได้บาง | ฉากทัศน์ 4 นูซันตาราบนฐานราก |

เพื่อให้ผู้อ่านสามารถเปรียบเทียบฉากทัศน์ทั้งสี่ได้อย่างเป็นระบบ รายงานกำหนดชุดตัวบ่งชี้ร่วมที่ทุกฉากทัศน์ต้องตอบเหมือนกัน ตัวบ่งชี้ชุดนี้ตั้งใจให้สะท้อนทั้งโครงสร้างเชิงระบบและประสบการณ์ชีวิตของเด็กกลุ่มเป้าหมาย โดยแยกบทบาทของอัตลักษณ์และความรู้สึกเป็นส่วนหนึ่งออกจากบทบาทเชิงเศรษฐกิจอย่างชัดเจน

| ตัวบ่งชี้ฉากทัศน์ (ใช้ร่วมกันทุกฉาก) | ความหมายที่ใช้ในบทนี้ |
|---|---|
| 1) เส้นทางการเรียนรู้หลักและการเชื่อมต่อ | เด็กเข้าถึงการเรียนรู้แบบใดบ้าง ทั้งในและนอกระบบ และสามารถเคลื่อนย้ายระหว่างเส้นทางต่าง ๆ ได้จริงเพียงใด โดยไม่ถูกตัดออกจากการรับรองสมรรถนะและโอกาสไปต่อ |
| 2) เศรษฐกิจครัวเรือนและภาระการดูแล | รูปแบบรายได้ของครัวเรือน ความมั่นคงของรายได้ และภาระดูแลที่มีผลต่อพื้นที่และเวลาที่เด็กใช้ในการเรียนรู้และพักผ่อน |
| 3) ตลาดงานและโอกาสรายได้ของเยาวชน | ประเภทของงานที่เปิดให้เยาวชนเข้าถึง ช่องทางเข้าสู่ตลาดงาน และความเสี่ยงการไหลออกของคนรุ่นใหม่จากพื้นที่ |
| 4) ความปลอดภัย ความไว้วางใจ และพื้นที่คุ้มครองเด็ก | ระดับความปลอดภัยในชีวิตประจำวัน ความไว้วางใจต่อสถาบันสาธารณะ และการมีพื้นที่ปลอดภัยที่เด็กกลุ่มเปราะบาง โดยเฉพาะเด็กกำพร้า สามารถเข้าถึงการคุ้มครองและการสนับสนุนได้จริง |
| 5) ทุนวัฒนธรรมและอัตลักษณ์ในฐานะพื้นที่ยืน | บทบาทของภาษา ศาสนา และวัฒนธรรมมลายูมุสลิมในฐานะฐานของความผูกพัน ความมั่นคงเชิงจิตใจ และเครือข่ายการช่วยเหลือ ไม่ว่าจะถูกใช้เป็นข้อจำกัดหรือเป็นจุดแข็ง |
| 6) การใช้ทุนวัฒนธรรมและอัตลักษณ์ในเศรษฐกิจ | วิธีที่ทุนวัฒนธรรมและอัตลักษณ์ถูกเปลี่ยนให้เป็นโอกาสทางเศรษฐกิจ เช่น งานสร้างสรรค์ การท่องเที่ยว หรือสินค้าและบริการฮาลาล โดยไม่ทำให้กลุ่มเปราะบางถูกกดดันหรือถูกใช้ประโยชน์อย่างไม่เป็นธรรม |
| 7) ดิจิทัลและความปลอดภัยออนไลน์ | การเข้าถึงโครงสร้างพื้นฐานดิจิทัลของเด็กและเยาวชน การใช้เทคโนโลยีในการเรียนรู้และหารายได้ และระบบคุ้มครองความปลอดภัยออนไลน์ |
| 8) ชีวิตของเด็กกำพร้าและเด็กกลุ่มเปราะบาง | ตำแหน่งของเด็กกำพร้าและกลุ่มเปราะบางในระบบการเรียนรู้ สวัสดิการ และโอกาสทางเศรษฐกิจ รวมถึงว่าพวกเขามีผู้ใหญ่หรือเครือข่ายคอยรับรองและพากลับเข้าสู่โอกาสเมื่อเกิดการสะดุดหรือไม่ |

ตัวบ่งชี้ทั้งแปดข้อนี้จะถูกใช้เป็น “เลนส์ร่วม” ในการอ่านฉากทัศน์แต่ละแบบ ผู้อ่านสามารถใช้กรอบดังกล่าวเพื่อเปรียบเทียบว่า ภายใต้โลกทั้งสี่แบบ เส้นทางชีวิตของเด็กและเยาวชน โดยเฉพาะผู้ที่มีความเปราะบางสูง จะยืนอยู่ตรงไหนของระบบ และเงื่อนไขใดที่จำเป็นต้องถูกเปลี่ยนแปลงเพื่อให้ภาพอนาคตที่พึงประสงค์ในช่วงท้ายบทกลายเป็นสิ่งที่ทำได้จริงในบทถัดไปของรายงาน
